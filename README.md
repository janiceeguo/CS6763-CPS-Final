# CS6763-CPS-Final

Data was obtained from the [UQ-AAS](https://github.com/xie00059/Amazon-Alexa-UQ-AAS21-datasets/tree/main) dataset using the 2021 files by Xie et al.

The subset of data we are using is compiled into `processed_alexa_skills.csv`, which selects around 95-140 random rows from the original data for each unique category based on popularity and ambiguity.
